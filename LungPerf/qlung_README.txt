              Quality Assurance of Quantitative Analysis of Lung Imaging


This disc contains the images of ten simulated lung studies. Each
study has two views of the lung, anterior and posterior in a 128x128
matrix. The data is presented in interfile version 3.3 format.




     Patient No.         Anterior filename   Posterior filename

     1                      QLUN01AN            QLUN01PO
     2                      QLUN02AN            QLUN02PO 
     3                      QLUN03AN            QLUN03PO 
     4                      QLUN04AN            QLUN04PO
     5                      QLUN05AN            QLUN05PO
     6                      QLUN06AN            QLUN06PO
     7                      QLUN07AN            QLUN07PO
     8                      QLUN08AN            QLUN08PO
     9                      QLUN09AN            QLUN09PO
     10                     QLUN10AN            QLUN10PO


Each interfile image is stored in two files, one containing the
header (.hdr) and one containing the image data (.img).

For example, patient 1 has four files containing the data for the
two views:

          QLUN01AN.hdr      header file for anterior
          QLUN01AN.img      image data for anterior
          QLUN01PO.hdr      header file for posterior
          QLUN01PO.img      image data for posterior
